# IoT Smart School Safety System

An IoT-based Smart School Safety System developed as our Graduation Project.

As the Team Leader, I led the project planning, system architecture, hardware and software integration, and coordinated the implementation process.

The system automates student attendance monitoring using RFID technology, ESP32 microcontrollers, Firebase Realtime Database, and a Flutter mobile application. It provides real-time attendance tracking, cloud synchronization, OLED status display, and instant notifications to parents.

## Technologies

- ESP32
- RFID (MFRC522)
- Flutter
- Firebase Realtime Database
- OLED Display
- Wi-Fi
- Arduino IDE

## Features

- Real-Time Attendance Monitoring
- School Gate Unit
- School Bus Unit
- Parent Mobile Application
- Instant Notifications
- OLED Display
- LED & Buzzer Feedback
- Cloud Database
