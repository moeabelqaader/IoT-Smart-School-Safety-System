# 🛡️ IoT Smart School Safety System

> **Graduation Project | Team Leader | October High Institute for Engineering & Technology**

An IoT-based Smart School Safety System developed as our Graduation Project to enhance student safety and automate attendance monitoring using **RFID technology**, **ESP32 microcontrollers**, **Firebase Realtime Database**, and a **Flutter mobile application**.

As the **Team Leader**, I was responsible for leading the project planning, system architecture, hardware-software integration, coordinating team tasks, and contributing to the implementation and testing phases.

---

# 📌 Project Overview

The system provides a real-time student monitoring solution by tracking attendance at both the **School Bus Unit** and the **School Gate Unit**.

Whenever a student scans an RFID card, the system instantly:

- Identifies the student
- Records attendance
- Updates Firebase Realtime Database
- Displays the status on the OLED screen
- Sends an instant notification to the parent

# 🔌 Hardware Components

| Component | Description |
|------------|-------------|
| ESP32 | Main Microcontroller |
| MFRC522 RFID Reader | Reads Student RFID Cards |
| RFID Card | Student Identification |
| OLED Display | Displays Student Status |
| Green LED | Valid Card Indicator |
| Red LED | Invalid Card Indicator |
| Buzzer | Audio Feedback |
| Wi-Fi | Cloud Communication |
